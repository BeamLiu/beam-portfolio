## 📋 资源准备优先级

### ⚠️ 第一优先级（必需）
- [ ] 简历 PDF 文件
- [ ] 个人社交链接（GitHub/LinkedIn/微信/邮箱）
- [ ] 更新数据文件中的个人信息

**参考文档**：[资源清单 - 必需资源](./resource_checklist.md#-必需资源高优先级)

### 🔵 第二优先级（推荐）
- [ ] AI 项目演示视频（EMOP 3.0、CAD 转换、StyleMidi）
- [ ] 项目截图
- [ ] 售前 PPT（PDF 格式）
- [ ] 早期极客项目视频

**参考文档**：[资源清单 - 视频资源](./resource_checklist.md#-视频资源推荐准备)

### ⚪ 第三优先级（可选）
- [ ] 个人头像照片
- [ ] 项目缩略图
- [ ] 英文翻译内容
- [ ] 额外的演示材料

**参考文档**：[补充内容说明](./additional_content.md)

## 🎯 内容准备时间规划

### 第一阶段：基础信息（1天）
- 准备简历 PDF
- 整理社交媒体链接
- 更新个人信息
- 测试基本功能

### 第二阶段：核心内容（3-5天）
- 录制/整理 AI 项目演示视频
- 准备项目截图
- 整理售前 PPT
- 上传资源并测试

### 第三阶段：完善优化（2-3天）
- 准备早期项目视频
- 添加个人头像
- 补充英文翻译
- 全面测试优化

**详细步骤**：参考 [资源清单 - 准备步骤](./resource_checklist.md#-准备步骤)

## 💡 使用技巧

### 准备内容时
1. 先查看 [资源清单](./resource_checklist.md) 了解需要什么
2. 使用 [内容模板](./content_template.md) 快速填充
3. 参考 [补充内容说明](./additional_content.md) 了解详细要求
4. 边准备边在清单上打勾

### 遇到问题时
1. 查看对应文档的"注意事项"或"提示"部分
2. 参考 [资源清单 - 资源准备技巧](./resource_checklist.md#-资源准备技巧)
3. 查看项目主目录的 `QUICK_START.md`

### 验证内容时
1. 使用 [资源清单 - 验证清单](./resource_checklist.md#-验证清单)
2. 运行 `npm run dev` 实时预览
3. 检查所有链接和资源
