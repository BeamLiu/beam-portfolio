## 📦 技术栈

- **构建工具**: Vite 7.3.1
- **框架**: React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **字体**: Google Fonts (Inter + JetBrains Mono)
- **状态管理**: Context API
- **国际化**: 自定义翻译系统

- **构建时间**: ~2.4 秒
- **组件数量**: 11 个
- **零错误、零警告**
